App Overview

IdeaSpark is a fully offline Progressive Web App that generates creative project ideas across science, entrepreneurship, social innovation, and interdisciplinary domains. It is built with HTML, CSS, and JavaScript — no backend, no login, no internet required after first load.

Section 02

Installing the App

Android (Chrome)
📱 Install on Android
  1. Open in Google Chrome on Android.
  2. After a few seconds, an "Install IdeaSpark" banner appears at the bottom.
  3. Tap Install. The app icon appears on your Home Screen.
  4. Alternatively: tap ⋮ menu → "Add to Home Screen".
Desktop (Windows / Mac)
🖥 Install on Desktop
  1. Open in Chrome or Edge.
  2. Click the install icon (⊕) in the address bar.
  3. Select Install. Opens as standalone app.
Local / Classroom Server
🏫 Run on School Network
  1. Place all files in one folder on a computer.
  2. Run: python -m http.server 8000
  3. Students connect via: http://[teacher-IP]:8000 on school Wi-Fi.
  4. Works fully offline — no internet needed once loaded.
💡

Free Online Hosting: Deploy to GitHub Pages or Netlify for a permanent shareable link. Netlify: drag-and-drop the folder → get a live URL in 30 seconds.

Using the Expand Full Outline Feature

After generating an idea, press "📋 Expand Full Outline" to reveal a complete project framework including Objective, Materials, Step-by-Step Process, and Expected Outcome. This gives students everything needed to start the project immediately.

🎓

Classroom Use: Project the Science tab on the board. Generate 3 ideas on Intermediate difficulty. Ask students to vote on their favourite and then use the Expand Outline to assign it as a class project over 2 weeks.

Section 06

Social Innovation Tab 🌱

Generates ideas for solving real community problems. Builds civic engagement and social awareness aligned with NCF 2023's social science goals.

Issue AreaProblems Addressed
🌿 EnvironmentPlastic waste, urban heat, water, biodiversity
📚 Education AccessLearning gaps, digital divide, library access
❤️ HealthHygiene awareness, mental health, rural diagnostics
♻️ Waste & EnergyComposting, solar power, e-waste, efficiency

The Scope selector (Local / District / National) adjusts the scale and ambition of the idea:

  • Local: Village or school-level project, achievable by 1–5 students in 1 month
  • District: Multi-school or panchayat-level initiative requiring teacher/parent support
  • National: Policy-level ideas suitable for essay competitions and youth forums

Each idea includes: Community Problem, Beneficiaries, Implementation Steps, and Expected Impact.

🌱

Classroom Use: Assign each student group a different issue area. Have them generate a Local idea and present a 5-minute implementation plan. This creates an instant service-learning project bank for the school.

Section 09

Classroom Use Guide

Suggested 45-Minute Session Plan

1
Warm-Up: Idea Shower (5 min)
Teacher generates 4 ideas using different modules on the projector. Students discuss which seems most interesting and why.
2
Individual Exploration (10 min)
Each student generates at least 3 ideas across any modules. They save their favourite.
3
Expand & Plan (15 min)
Students expand their saved idea using the Full Outline feature. They write a 1-page project plan in a notebook.
4
Combine Challenge (10 min)
Teacher assigns random domain pairs to groups. Each group generates and presents their combined idea in 2 minutes.
5
Reflection (5 min)
Discussion: Which idea is most relevant to our community? How does this connect to NEP 2020's vision of creative, innovative learners?

Long-Term Student Projects

IdeaSpark works best as the starting point for real student projects spanning several weeks:

  • Week 1: Generate and save 5 ideas. Choose one for a project.
  • Week 2: Expand the outline. List required materials. Make an action plan.
  • Week 3–4: Implement the project. Document with photos/notes.
  • Week 5: Present findings to class. Connect back to the project idea card.
Section 10

Troubleshooting

IssueSolution
App not working offlineOpen once with internet to trigger caching. After that, works fully offline.
Install prompt not showingUse Google Chrome or Edge. Safari (iOS) has limited PWA support.
"Try a different combination" messageSome category + difficulty combinations have fewer entries. Try "All" category or a different difficulty.
Saved ideas disappearedSaved ideas use browser LocalStorage. Clearing browser data removes them. Export by copying to notes app first.
Photo not showingEnsure author.jpg is in the same folder as index.html. App shows a placeholder if not found.
Console error (SW 404)The app uses an inline service worker fallback. This error is harmless in preview — works normally when hosted.
