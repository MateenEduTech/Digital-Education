1️⃣ Sequence — Do things in order ▼

A sequence is a set of instructions that run one after another in a specific order. This is the most basic concept in programming.

Step 1: Wake up
Step 2: Brush teeth
Step 3: Have breakfast
Step 4: Go to school
→ Each step happens in ORDER

In code, every line executes in sequence unless told otherwise. Changing the order changes the result!

🔁 Loop — Repeat actions ▼

A loop repeats a block of instructions multiple times. Instead of writing the same step over and over, you use a loop!

REPEAT 5 times:
  → Print "Hello World"

// Result: Hello World (5 times)
// Much better than writing it 5 times!

Types of loops: FOR loop (repeat a set number of times), WHILE loop (repeat while a condition is true).

❓ Condition — Make decisions ▼

A condition (IF-ELSE) lets your program make decisions based on whether something is true or false.

IF it is raining:
  → Take umbrella
ELSE:
  → Wear sunglasses

// The program chooses a path based on the condition!

⚙️ Function — Reusable blocks ▼

A function is a named block of steps you can reuse. Instead of repeating the same instructions, you define them once and call the function whenever needed.

FUNCTION makeTea():
  Boil water
  Add tea bag
  Pour in cup
  Add sugar

makeTea() // Call the function!
makeTea() // Use it again anytime!

Functions make code reusable, organized, and easier to understand.
